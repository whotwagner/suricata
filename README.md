# Suricata

This gem offers classes to handle suricata logfiles. It ships with a nagios-plugin and an analysis tool.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'suricata'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install suricata
